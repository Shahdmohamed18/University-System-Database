# University-System-Database
A university database project designed to manage academic data efficiently. The project includes database design (ERD and Schema), simple and complex SQL queries, a basic web-based GUI using HTML, and PHP integration for database connectivity and operations.
University Database Management System
Overview

This project is a University Database Management System developed as part of a database course project. It demonstrates database design, implementation, and interaction through a simple web-based interface.

Features
Database design using ERD and Schema.
Simple SQL queries.
Complex SQL queries.
Basic GUI developed with HTML.
PHP integration for database connectivity and operations.
Project Structure
SQL scripts for database creation and queries.
ERD and Schema diagrams.
HTML files for the user interface.
PHP files for backend functionality.
Technologies Used
MySQL
SQL
PHP
HTML
